Air List

Created a website that visualized AirBnB listing data in San Francisco. Additional features included a cost and revenue estimator based off location. AirBnB data provided by Capital One.

Placed top 40 in Capital One's MindSumo challenge; was invited to their Software Engineering Summit for Winter 2018.

See it live on Heroku here. The GitHub Repo can be found here.

  • Location: Wherever I happened to be when I had free time
  • Date: Fall 2017
  • Languages: JavaScript, NodeJS, HTML/CSS
Freebie Magnet

Created a website that connected the user to events (with free stuff!) near them. How it works: Organizations log into the site, and add events through a form. Users would then be able to view events near them.

See it live at FreebieMag.net. Created during HackUMBC Fall 2017. Devpost can be found here.

  • Location: JHU Medical Campus
  • Date: Fall 2017
  • Languages: JavaScript (w/ Firebase), HTML/CSS
MedLog

Created a prototype Android app that identifies a prescription pill through the camera, and warns the user of any potential side effects with their currently logged medications. The user can then decide whether or not to add the pill to their log.

Created during the Johns Hopkins Medical Hackathon (MedHacks). My team placed top 15 and recieved the Wolfram Alpha Award. Devpost can be found here.

  • Location: JHU Medical Campus
  • Date: Fall 2017
  • Languages: Java, Android Studio
Password Data Visualization

As part of the NIST Summer High School Intern Program, I developed a set of programs to analyze and visualize various sets of password data. Graphing scripts used the d3.js library and open source starter code from Mike Bostock.

Code can be found on the NIST github.

  • Location: NIST ITL
  • Date: Summer 2016
  • Languages: Java, Javascript
